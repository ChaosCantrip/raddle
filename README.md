# RaDdle [[raddle.online]](https://raddle.online)

Raddle is a simple guessing game inspired by Wordle. Players guess the Destiny 2 Raid or Dungeon Encounter for each day, or can play freely in Arcade Mode.

RaDdle was created using `next.js`, and uses `mongodb` in the backend.

RaDdle will eventually become part of the ChaosD2 Project, when I get all of that up and running again.
